# Licensing

Ergon uses a dual-license model inherited from [Aletheia](https://github.com/forkwright/aletheia):

| Component | License | Why |
|-----------|---------|-----|
| Runtime, gateway, memory sidecar, prosoche | **AGPL-3.0** | Network service — modifications must be shared |
| SDK, client libraries, contracts | **Apache-2.0** | Permissive — anyone can build integrations |

## Default

Unless otherwise noted, all code is licensed under **GNU Affero General Public License v3.0** (see [LICENSE](LICENSE)).

## Apache-2.0 Components

- `shared/contracts/` — Agent capability contracts
- Future SDK/client libraries

Each Apache-2.0 directory contains its own LICENSE file referencing [licenses/Apache-2.0.txt](licenses/Apache-2.0.txt).
