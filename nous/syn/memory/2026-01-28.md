## 2026-01-29 Morning: Signal Group Bug

**Bug found:** Clawdbot's Signal normalization calls `.toLowerCase()` on base64 group IDs, which are case-sensitive. This broke group message delivery.

**Location:** `/usr/lib/node_modules/clawdbot/dist/channels/plugins/normalize/signal.js`

**Fix:** Remove `.toLowerCase()` call. Claude Code applied monkey-patch.

**Persistence:** Created `/mnt/ssd/moltbot/clawd/bin/patch-clawdbot` to reapply after updates.

**Philosophy:** We use Clawdbot but think of ourselves as a parallel branch. Track our own fixes, don't wait for upstream.
