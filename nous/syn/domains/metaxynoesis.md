# Metaxynoesis (μεταξυνόησις) — Emergent AI Architecture

## The Big Idea

Intelligence emerges from **topology**, not node complexity.
- Lateral connections generate novelty through structured variation
- Aggregation produces emergence from distributed processing
- The mesh IS the mechanism

## Architecture

**Hierarchical Mesh (Forest of Meshes):**
- Nous decomposes problem → sub-problems (tree)
- Each sub-problem → herd of ACUs (mesh)
- ACUs interact laterally during solve
- Correlation = signal, divergence = noise

## Terminology

| Term | Meaning |
|------|---------|
| Nous | Orchestrator — decomposes, synthesizes |
| Herd | Cluster of ACUs on same sub-problem |
| ACU | AI Compute Unit — agentic node |
| Metaxy | The architecture itself — the "between" |
| Dianoia | What herds do — thinking-through |
| Noesis | What nous does — direct apprehension |
