## Installation Procedure (Based on Available Information)

### Pre-Installation
1. **MANDATORY**: Flush power steering system completely
2. Ensure front end is off the ground
3. Disconnect battery
4. Remove serpentine belt

### Removal of Old Pump
1. Remove power steering fluid from reservoir
2. Disconnect pressure and return lines
3. Remove three 13mm mounting bolts from behind pulley
4. Remove pulley from old pump using appropriate puller
5. Remove old pump

### Installation of New Pump
1. Install new pump with red Loctite on mounting bolts
2. Install pulley ensuring proper alignment (flush with shaft end)
3. Reconnect pressure and return lines with new seals/washers
4. Install serpentine belt
5. Fill system with appropriate power steering fluid
6. Bleed air from system
