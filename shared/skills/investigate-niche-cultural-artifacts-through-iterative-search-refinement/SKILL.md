# Investigate Niche Cultural Artifacts Through Iterative Search Refinement
Systematically search for information about obscure design elements in cultural artifacts by progressively refining queries based on initial findings.

## When to Use
When trying to understand or identify mysterious visual or symbolic elements in niche cultural products (albums, artwork, merchandise) where direct searches yield limited results, and community forums may contain expert discussion.

## Steps
1. Enable web search and web fetch tools
2. Attempt direct search with specific identifying details (artist name, product name, element description)
3. If initial search fails, broaden the query to focus on the artifact itself rather than the specific element
4. Search for related keywords that might describe the mysterious element (visual characteristics, category guesses)
5. When search results mention community discussions, fetch those pages directly (especially Reddit threads in relevant subreddits like r/Symbology)
6. Refine search queries based on any clues found in community discussions
7. Try multiple search angles combining different possible interpretations of the element (e.g., "cattle brands" vs "border symbols")
8. Fetch JSON versions of Reddit threads for more complete data extraction

## Tools Used
- web_search: iteratively refine queries based on previous results and community hints
- web_fetch: access full content of promising community discussion threads and extract detailed responses
