# Investigative Research Through Multi-Source Web Gathering
Systematically search for and fetch content from multiple web sources to build a comprehensive understanding of a specific topic or controversy.

## When to Use
When you need to investigate a complex topic, controversial issue, or current event that requires gathering information from multiple sources to understand different perspectives and find detailed reporting.

## Steps
1. Enable web_search tool
2. Conduct initial broad web searches using key terms related to the topic
3. Conduct follow-up searches with refined or alternative query terms to find additional sources
4. Enable web_fetch tool
5. Fetch full content from the most relevant search results to get detailed information
6. Fetch content from additional sources discovered through search results
7. Conduct targeted searches combining key entities, locations, dates, and specific details from initial findings to uncover related incidents or developments

## Tools Used
- web_search: locate relevant articles and sources on the topic with multiple query variations
- web_fetch: retrieve full article text from URLs for detailed information and verification
