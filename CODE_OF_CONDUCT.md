# Code of Conduct

## Standard

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Summary

- Be respectful and constructive
- No harassment, discrimination, or personal attacks
- Focus on technical merit
- Assume good intent

## Scope

Applies to all project spaces: issues, PRs, discussions, and any public representation of the project.

## Enforcement

Report violations to the maintainers via [GitHub Security Advisories](https://github.com/forkwright/aletheia/security/advisories/new) or by opening a private issue.

Maintainers will review and respond within 7 days. Actions may include warnings, temporary bans, or permanent bans depending on severity.
